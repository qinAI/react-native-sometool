
# react-native-sometool

```
这并不是一个框架,只是个人写的一个例子.
上传到npm的部分一句代码也没有.
只是学习怎么发布框架.
sometooldome 文件夹中写了一些小东西.


由于react-native-video-processing的
iOS    部分无法正常运行SWIFT版本问题,所以使用了视频处理部分RNVideoTrimmer
```
